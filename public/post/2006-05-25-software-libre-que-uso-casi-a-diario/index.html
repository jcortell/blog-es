Software Libre que uso casi a diario

25, 2006 Free Software, General, Personal, Technology

Un amigo total noob, harto de leer y leer artí­culos sobre software libre, y que cree que install, make, makeinstall es similar a abracadabra patadecabra, o a in nomine pater me ha pedido que le dé ejemplos de software libre que uso a diario o casi. Los “fundamentales”.

Ya que he recopilado la lista para él, pues ya puestos la comparto con vosotros -¿no? (y así­, además, seguro que se generan un montón de comentarios interesantes con otros programas a tener en cuenta).

Nota.- Yo uso hardware Apple Macintosh, y como OS voy cambiando entre MacOSX y Ubuntu GNU/Linux. Pero mi amigo es Mac/Mac, así­ que el software libre (en algún caso simplemente freeware) que voy a listar es para Mac OSX (aunque la mayorí­a tiene versiones GNU/Linux y Windows).

  • FTP: CyberDuck
  • PIM, PDR, Messaging, Groupware: Aethera
  • Reproductor multimedia: XinePlayer
  • Emulación: QEmu (podeí­s instalar de todo)
  • Emulación Mac: Mac-on-Mac (por ejemplo para correr OS9 en mi MacBookPro MacIntel)
  • Wine (no emulación): DarWine
  • Editor HTML WYSIWYG: NVU
  • Ilustración vectorial: Inkscape
  • Maquetación: Scribus
  • Edición y retoque de imágenes: the GIMP
  • Procesador de documentos: LyX
  • Suite Ofimática: OpenOffice
  • Lanzadera multiutilidad: QuickSilver
  • Antivirus (no nos hace falta a los Mac/GNULinux, pero por si acaso): ClamXav
  • Copias de Seguridad: Carbon Copy Cloner
  • VoIP: Gizmo (no es del todo libre, pero sus estándares sí­, a diferencia de Skype)
  • Blog: WordPress
  • Navegador: Shiira
  • RSS: Vienna
  • Network Monitor: AquaEthereal
  • P2P: BitsOnWheels (BitTorrent) / Cabos (Gnutella)
  • Copias Privadas (de DVDs): HandBrake (vaya, según el código penal, por incitaros a ejercer vuestros derechos con esta tecnologí­a podrí­a ir a la cárcel 2 años… pues que le den a un código penal que atenta contra nuestros derechos)

Otros que no uso tan frecuentemente, pero que están muy bien:

  • Podcasting: Juice
  • Desktop: KDE 4 en MacOSX
  • Gentoo: Gentoo for MacOSX (cuidado: alpha.- leer este howto)
  • Juegos (muchos de ellos sólo freeware): MAME (emulador) / Cube / Marathon / GoBan / Wolfenstein Enemy Territory / GrandTheftAuto 1 y 2 / Armagetron Advanced / TuxRacer / Frozen-Bubble
  • MMORPGs: Entropia / Battle for Wesnoth

Iré ampliando la lista a medida que se me ocurran / descubra nuevo software.
